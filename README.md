# Portfolio

Personal portfolio website for Aashrit Surapaneni.

## Live Site

https://aashrit.link/

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages
- AWS Route 53 (DNS)
- Custom domain: aashrit.link

## Structure

```
Portfolio/
├── index.html
├── about/index.html
├── projects/index.html
├── contact/index.html
├── styles.css
├── script.js
├── favicon.svg
├── Image.jpeg
└── CNAME
```

## Pages

- `/` - Home
- `/about/` - About and skills
- `/projects/` - Projects showcase
- `/contact/` - Contact information

## Deployment

- Hosted on GitHub Pages
- Custom domain configured via Route 53
- HTTPS enabled via Let's Encrypt
- Deployed from `main` branch

## Local Development

```bash
git clone https://github.com/aashrit01/Portfolio.git
cd Portfolio
python -m http.server 8000
```

Visit `http://localhost:8000`
