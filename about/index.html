About Me

Hello! I'm Aashrit Surapaneni, a Machine Learning Engineer experienced in designing, building, and deploying AI solutions across healthcare, finance, and enterprise platforms. My work bridges classical ML and modern GenAI systems, with a strong focus on scalable pipelines, MLOps, and production-grade applications.

Currently, I'm focused on building LLM-based applications with Retrieval-Augmented Generation (RAG) and Function Calling, deploying real-time AI APIs using FastAPI, Docker, and Kubernetes, and automating document processing using Azure Form Recognizer and OpenAI GPT-4-turbo.

I'm passionate about extracting insights from structured and unstructured data using hybrid pipelines with OCR, spaCy, and prompt engineering. Always exploring what's next in intelligent systems, agentic workflows, and real-world LLM deployment.
